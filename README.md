# AUTOMAÇÃO E ANÁLISE DE DADOS 

Este repositório reúne projetos desenvolvidos durante um intensivão de lives promovidaas pelo canal HASHTAG PROGRAMAÇÃO, com foco em **Python**, automação de processos e análises de dados

## OBJETIVO
Aplicar conceitos práticos de programação para resolver problemas reais, desenvolver raciocínio analítico e aprimorar habilidades técnicas voltadas ao mercado de tecnologia e negócios

## SOBRE O PROJETO
O intensivão foi dividido em 4 aulas, cada uma com foco em uma área específica:
     - Automação de tarefas com Python
     - Análise e tratamento de dados
     - Criação de relatórios e insights
     - Chat Bot com aplicações de Inteligência Artificial

Durante o processo, desenvolvi: 
   **3** projetos completos
   **1** projeto em andamento (dependente de acesso a ferramenta de IA paga)

   
## TECNOLOGIAS UTILIZADAS
    - Python
    - Bibliotecas para análise de dados
